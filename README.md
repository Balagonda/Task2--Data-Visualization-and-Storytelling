# Task 2 - Data Visualization and Storytelling

## Project Overview
This project focuses on creating an interactive Power BI dashboard using the Sample Superstore dataset. The objective is to transform raw sales data into meaningful visualizations and business insights that support data-driven decision-making.

## Objective
- Analyze sales and profit performance.
- Create interactive visualizations using Power BI.
- Identify key business trends and patterns.
- Present findings through data storytelling.

## Tools Used
- Power BI Desktop
- Microsoft Excel (for data review)
- GitHub

## Dataset
Dataset Used: Sample Superstore Dataset

The dataset contains information about:
- Orders
- Sales
- Profit
- Customers
- Products
- Categories
- Regions
- Order Dates

## Dashboard Components

### KPI Cards
- Total Sales
- Total Profit
- Total Orders

### Visualizations
1. Sales by Category
2. Profit by Category
3. Monthly Sales Trend
4. Sales by Region
5. Top 10 Customers by Sales
6. Profit by Sub-Category

### Filters/Slicers
- Region
- Category
- Order Date

## Key Insights

### Sales Performance
- Total Sales reached approximately 2.30 Million.
- A total of 9,994 orders were processed.

### Category Analysis
- Technology generated the highest sales and profit.
- Furniture showed comparatively lower profitability.

### Regional Analysis
- West region contributed the highest share of sales.
- Sales distribution across regions was relatively balanced.

### Customer Analysis
- A small group of customers contributed significantly to total revenue.
- Identifying and retaining high-value customers can improve business growth.

### Product Analysis
- Copiers and Phones were the most profitable sub-categories.
- Certain sub-categories require attention to improve profitability.

### Trend Analysis
- Monthly sales trend showed fluctuations over time.
- Seasonal patterns and declining periods can be analyzed for future planning.

## Business Recommendations
- Increase focus on Technology products due to strong performance.
- Strengthen customer retention programs for top customers.
- Improve marketing efforts in high-performing regions.
- Review low-profit product categories and optimize pricing strategies.
- Monitor monthly sales trends to identify growth opportunities.
